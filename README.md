## Задание 1
### Найдите внутри C# скрипта “коэффициент корреляции ” и сделать выводы о том, как он влияет на обучение модели.
Коэффициент корреляции — это статистическая мера, которая показывает степень взаимосвязи между двумя переменными. В контексте машинного обучения он может использоваться для оценки зависимости между входными признаками и целевой переменной.
Внутри скрипта Move есть переменная tempInf, которая определяет порог инфляции. Если агент не позволяет инфляции подняться выше порога (в нашем случае 6%), он получает положительное вознаграждение, иначе отрицательное.


## Задание 2
### Изменить параметры файла yaml-агента и определить какие параметры и как влияют на обучение модели. Привести описание не менее трех параметров.

- max_steps: максимальное количество шагов для завершения обучения. Чем выше число, тем лучше будет обучаться агент.
- learning_rate: скорость обучения. При высоких показателях агент начинает обучаться некорректно, а при более низких обучение будет проходить медленно.
- num_epoch - количество эпох при обучении модели.







## Задание 3
### Приведите примеры, для каких игровых задачи и ситуаций могут использоваться примеры 1 и 2 с ML-Agent’ом. В каких случаях проще использовать ML-агент, а не писать программную реализацию решения?
Использование ML-агентов в игровых задачах может быть весьма эффективным, особенно в ситуациях, где традиционные алгоритмы программирования могут оказаться сложными или неэффективными.
ML-агента проще использовать, когда задача требует сложного поведения, которое трудно запрограммировать вручную, а также когда необходимо быстро протестировать различные стратегии и оптимизировать поведение агента без написания большого количества кода.
